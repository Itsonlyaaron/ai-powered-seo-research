# AI-Powered SEO Content Production Research

## Overview

This repository contains research on AI-powered SEO content production for B2B SaaS. The objective is to study how leading SEO professionals leverage AI to improve content creation, optimization, keyword research, and organic search performance.

The project collects and organizes insights from LinkedIn posts, YouTube videos, podcasts, and other trusted industry resources.

---

## Objectives

- Identify 10 industry experts in AI-powered SEO.
- Collect their latest LinkedIn content.
- Collect YouTube transcripts.
- Analyze podcast discussions.
- Document practical SEO workflows.
- Organize all research in a structured GitHub repository.
